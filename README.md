# üíç Wedding Invite Microsite

A mobile-first wedding invitation website with reel-inspired animations.

## Features
- Hero section with looping background video
- Scroll-triggered animations using GSAP
- Wedding events cards
- Venue information with map link
- Background music toggle
- Fully responsive design

## Setup

1. Add your assets:
   - Place your video file in `assets/video/hero.mp4`
   - Place your music file in `assets/music/bg-music.mp3`

2. Customize content in `index.html`:
   - Names
   - Date
   - Location
   - Events

3. Open `index.html` in a browser to test locally

## Deployment to GitHub Pages

1. Create a new GitHub repository
2. Push all files to the `main` branch
3. Go to Settings ‚Üí Pages
4. Select branch `main` and folder `/root`
5. Save and wait for deployment
6. Share the generated URL with guests

## Customization

- **Colors & Fonts**: Edit `style.css`
- **Content**: Edit `index.html`
- **Animations**: Adjust timing in `script.js`

## Tech Stack
- HTML5
- CSS3
- JavaScript
- GSAP + ScrollTrigger
- GitHub Pages
